Overview : Dunder Mifflin Co. Customer Contact Website
This is my Final year of University advance web development conisting of building a responsive website including CSS, Html, and PHP language. 
The website is about a paper company named Dunder Mifflin Co. built to help customers easily get in contact with the compnay, request quotes , and learn about available paper products and services.
The site emphasizes simplicity and clear contact functionality for B2B and retail clients. 

Features:
Fully responsive design (desktop, mobile, tablet)
Contact form with form validation and email integration
Easy navigation and smooth scrolling 
Employee information page available 
Customer inquiry confirmation page

Tech Stack
Frontend: Html, CSS5, Javascript
Backend: PHP
Database: MySQL
other tools: git, VS Code, XAMPP, FileZilla, localhost
